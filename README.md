# n-bot

A Binance trading bot and analysis toolset.

## Features
- Candle data fetching
- Technical indicators
- PDF report generation
- Trend analysis
- Visualization tools

## Setup
1. Install dependencies (listed in scripts or required by Python files).
2. Configure your API keys (not included in this repository).
3. Run `run.py` to start.

## Files
- `run.py`: Main entry point.
- `fetch_candles.py`: Utility for downloading market data.
- `indicator.py` / `indicatoros.py`: Technical analysis implementations.
- `plt_data.py`: Plotting and data visualization.
- `get_pdf_coins.py`: PDF generation logic.
